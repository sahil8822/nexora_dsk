# Changelog

## 3.0.0 (The Intelligence Update)
* **Smart Vision:** Integrated Google ML Kit for native background face and barcode detection.
* **Audio Analysis:** Added native FFT (Fast Fourier Transform) for real-time frequency spectrum analysis.
* **Hardware Logging:** Automated background telemetry recording to CSV with buffered I/O.
* **Geofencing:** Added native background GPS triggers for intelligent location awareness.
* **Performance:** All AI and Signal processing occurs on native background threads to minimize memory load and UI lag.

## 2.5.0 (The Ultimate Update)
* **Biometrics Pro:** Added secure FaceID and Fingerprint authentication support.
* **Audio Pro:** Real-time raw PCM audio streaming from microphone.
* **Tactile Feedback:** Added Haptic Feedback patterns and custom vibration support.
* **Hardware Health:** Real-time Battery level, Charging status, and Thermal monitoring.

## 2.4.0 (The Pro Update)
* **Camera Pro:** Added support for Flash/Torch, Digital Zoom, and Camera Flipping.
